Guide d’installation du programme

1ère étape : prendre le dossier du projet.

2ème étape : ouvrir le dossier et faire un double click sur le fichier main(main.py).

3ème étape : s’assurer que votre éditeur de code utilise la version 3.13.12 de python 

Exemple : certaines parties de codes ne fonction pas en 3.9.

4ème et dernière étape : s’assurer que le dossier des images s’appelle « image » pour éviter tout problème.
