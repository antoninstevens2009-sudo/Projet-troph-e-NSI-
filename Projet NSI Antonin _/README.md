Guide d’installation du programme

1ère étape : prendre le dossier du projet.

2ème étape : ouvrir le dossier et faire un double click sur le fichier main(main.py).

3ème étape : s’assurer que votre éditeur de code utilise la version 3.13.12 de python 

Exemple : certaines parties de codes ne fonction pas en 3.9.

4ème et dernière étape : s’assurer que le dossier des images s’appelle « image » pour éviter tout problème.  
 Voici comment utiliser le programme:
 Ce projet est un jeu développé en Python avec la bibliothèque Pygame. Il regroupe six mini-jeux différents. Le but est de tous les réussir afin de débloquer les pièces d’un puzzle et compléter l’image finale.

Lorsque le joueur lance le programme, il arrive sur un menu principal. Il peut cliquer sur l’un des six jeux pour le lancer, accéder aux règles ou ouvrir le puzzle. Chaque jeu peut être joué dans n’importe quel ordre.

Le jeu 1 est un jeu de calcul mental composé de 10 niveaux. Le joueur doit cliquer avec la souris sur la bonne réponse parmi plusieurs propositions. Il dispose de 3 vies. À chaque erreur, il perd une vie. S’il atteint le niveau 10, il gagne.

Le jeu 2 est un jeu d’exploration avec 3 niveaux. Le joueur se déplace avec les touches ZQSD ou les flèches directionnelles. Il doit éviter les ennemis et récupérer des coffres. Lorsqu’il est sur un coffre, il doit appuyer sur la barre ESPACE pour déclencher un calcul, puis entrer la réponse au clavier et valider avec la touche ENTRÉE. S’il touche un ennemi, il perd. S’il réussit les 3 niveaux, il gagne.

Le jeu 3 comporte 3 manches. Le joueur doit compter le nombre d’arbres affichés à l’écran avant la fin du chronomètre (10 secondes). Il doit ensuite entrer sa réponse au clavier et appuyer sur ENTRÉE pour valider. Si la réponse est correcte, il passe à la manche suivante. Après 3 manches réussies, il gagne.

Le jeu 4 est un pierre-feuille-ciseaux contre l’ordinateur. Le joueur choisit en cliquant avec la souris sur un bouton. Le premier à atteindre 5 points gagne la partie. En cas de victoire, le jeu est validé.

Le jeu 5 est un Pong. Le joueur contrôle la barre avec les touches flèche HAUT et BAS. Le but est d’atteindre 5 points avant l’ordinateur. Si le joueur gagne, le jeu est validé.

Le jeu 6 est un jeu inspiré de Flappy Bird. Le joueur doit appuyer sur ESPACE ou flèche HAUT pour faire monter le personnage. Il doit éviter les obstacles. Le but est d’atteindre un score de 8 pour gagner. Si le joueur touche un obstacle ou sort de l’écran, il perd.

Chaque jeu réussi débloque une partie du puzzle. Le joueur peut consulter le puzzle à tout moment depuis le menu. Les parties non débloquées sont affichées en rouge.

La progression est sauvegardée automatiquement dans un fichier, ce qui permet de garder les jeux déjà réussis même après avoir quitté le programme.

Une fois les six jeux gagnés, le puzzle est complété et un écran de victoire s’affiche.
